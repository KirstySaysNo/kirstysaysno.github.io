# kirstysaysno.github.io
### Headings
Use `#` for headings, with additional `#` increasing the heading level.

```markdown
# H1
## H2
### H3
#### H4
```

### Emphasis
- **Bold**: `**bold**` or `__bold__`
- *Italic*: `*italic*` or `_italic_`
- ***Bold and Italic***: `***bold and italic***`

### Lists
- **Unordered List**:
  ```markdown
  - Item 1
  - Item 2
    - Sub-item
  ```

- **Ordered List**:
  ```markdown
  1. First item
  2. Second item
```

### Links and Images
- **Link**: `[link text](url)`
- **Image**: `![alt text](image_url)`

### Code
- **Inline Code**: Use backticks: `` `code` ``
- **Code Block**:
  ```markdown
  ```language
  Code here
  ```
  ```

### Blockquotes
```markdown
> This is a quote.
```

### Tables
```markdown
| Header 1 | Header 2 |
|----------|----------|
| Row 1    | Data     |
| Row 2    | Data     |
```

### Task Lists
```markdown
- [ ] Task not done
- [x] Task done
```

### Horizontal Line
```markdown
---
```

### Footnotes
```markdown
Here is a sentence with a footnote.[^1]

[^1]: This is the footnote text.
```
